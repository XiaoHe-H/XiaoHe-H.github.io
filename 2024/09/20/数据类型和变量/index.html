数据类型和变量

数据类型

数据类型分为基本数据类型和引用数据类型

  • 基本数据类型
    • 整型
      • byte
      • short
      • int
      • long
    • 浮点型
      • float
      • double
    • 字符类型
      • char
    • 布尔类型
      • boolean
  • 引用数据类型
    • 接口(interface)
    • 数组([])
    • 类(class)
数据类型 大小 默认值
byte 1字节8位 0
short 2字节16位 0
int 4字节32位 0
long 8字节64位 0L
float 4字节32位 0.0F
double 8字节64位 0.0
char 2字节16位 ‘\u0’
boolean false

类型转换

转化从低级到高级：byte,short,char（三者同级）—> int —> long—> float —> double
低级转换高级：自动类型转换
高级转换低级：强制类型转换

自动类型转换
  • 数据类型相互兼容(整型和浮点型)
  • 目标类型大于源类型
强制类型转换

数据取值范围大的转换为数据取值范围小到的

now是int类型，但是rise是double，目标数据小于源数据，要进行强制转换，转换的方法在被赋值前面+(int)

变量

变量的命名规则

字下美人数骆驼

字母

下划线

美人：$

数：数字

骆驼：驼峰命名法

变量的首字母小写，如果有第二个单词，则第二个单词的首字母大写。

注意点：数字不能做为变量的开头

变量声明及使用

  • 声明变量，根据数据类型在内存申请空间
    • 数据类型 变量名;
  • 赋值(将数据存储到对应的内存空间)
    • 变量名 = 数值;
  • 简而言之
    • 数据类型 变量名 = 数值;
  • 使用变量

常量

固定不变的值，不能改变的值

  • 常量通常大写
  • 不同字符使用下划线分隔
  • 只能被赋值一次，通常定义时即对其初始化
  • 比较安全

运算符

算术运算符

  • 除法运算，两个操作数是整型的，结果也会是整型的，舍弃掉小数部分；如果有一个数是浮点数，结果将自动转型为浮点型
  • 取余运算，两个操作数是整型的，结果也会是整型的，如果有一个数是浮点数，结果将自动转型为浮点型
  • 自增自减运算，a++ 相当于 a = a + 1, a++ 是先运用在计算，++a 先计算在运用

关系运算符

  • “ = “是赋值运算，” == “是等于运算
  • “ > “、” < “、” >= “、” <= “只支持数值类型的比较，” == “、” != “支持所有数据类型的比较
  • “ > “、” < “、” >= “、” <= “运算符优先级别高于
  • 关系表达式的运算结果是布尔值

逻辑运算符

  • 优先级别 ! > && > ||
  • 操作数只能是布尔型，操作结果也是布尔型
  • & 和 && 的区别： && 左边是false时，不计算右边的表达式，左假即假； & 无论左边真假都会计算右边的表达式

赋值运算符

赋值符号 “ = “
语法：变量名 = 表达式；
可以和算术运算符结合成复合赋值运算符，例如” += “、” -= “、” *= “、” /= “、” %= “

条件运算符

三元/目运算符

条件 ? 表达式1 : 表达式2

如果条件为true，则返回表达式2的值，如果条件为false，则返回表达式3的值

运算符优先级

单目运算符包括 ! ++ –，优先级别高

优先级别最低的是赋值运算符

可以通过()控制表达式的运算顺序，()优先级最高

从右向左结合性的只有赋值运算符、三目运算符和单目运算符

算术运算符 > 关系运算符 > 逻辑运算符 > 赋值运算符

总结

变量的概念和使用

  • 概念

    在程序中存储数据并赋值

  • 使用

    数据类型+变量名 = 数值;

数据类型转换包括自动类型转换和强制类型转换

  • 基本数据类型 存储在栈区

    byte

    short

    int

    long

    float

    double

    char

    boolean

  • 引用数据类型 存储在堆区

    类

    接口

    数组

  • 转换的原则

​ 转化从低级到高级：byte,short,char（三者同级）—> int —> long—> float —> double

  • 自动类型转换

    数据类型相互兼容(整型和浮点型)

    当目标类型大于源类型

  • 强制类型转换

    数据取值范围大的转换为数据取值范围小到的

    转换的方法在被赋值前面+(要转换的数据类型)

Scanner类可以从键盘获取输入的信息

  • new一个Scanner类

  • 获取键盘数据

赋值运算符、算术运算符、关系运算符、逻辑运算符

这些运算符的优先级

算术运算符 > 关系运算符 > 逻辑运算符 > 赋值运算符
