使用记事本编写程序

使用记事本编译.java源程序。使用javac命令编译，生成.class文件(字节码文件)，使用java命令运行.class文件

JDK与JRE、JVM三种的关系

JDK

Java标准开发包，它提供了编译、运行Java程序所需的各种工具和资源，包括Java编译器、Java运行环境，以及常用的Java类库等。JDK里面包含JRE

bin

可执行文件

jre

运行java程序所依赖的JVM

lib

运行java程序所依赖的Jar包

JRE

Java运行环境，用于解释执行Java的字节码文件。JRE包含JVM

JVM

Java虚拟机，是JRE的一部分。它是整个java实现跨平台的最核心的部分，负责解释执行字节码文件，是可运行java字节码文件的虚拟计算机。所有平台的上的JVM向编译器提供相同的接口，而编译器只需要面向虚拟机，生成虚拟机能识别的代码，然后由虚拟机来解释执行。

使用Java编译器编译Java程序时，生成的字节码文件(与平台无关)，.class文件只面向JVM，都提供了相同的接口。JVM是Java程序跨平台的关键部分

区别与联系

JDK 用于开发，JRE 用于运行java程序。

JDk包含JRE，JDK 和 JRE 中都包含 JVM。

JVM 是 java 编程语言的核心并且具有平台独立性。

简述使用记事本开发Java程序的步骤

  1. 新建文本文档，后缀名改为.java文件
  2. 在文本中编写java程序代码
  3. 打开终端运行java文件
    • Win+R输入cmd，将路径改到你的.java文件下
    • javac空格，再输入文件名.java，回车(enter键)，此时编译器会生成对应的字节码文件
    • java 输入文件名，进行编译运行
